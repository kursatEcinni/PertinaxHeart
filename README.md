DeskHub — Smart Desktop Hub

Overview
- ESP32-S3 + ST7789 (240x240) display, rotary encoder input, DHT22 + distance sensor, Wi‑Fi, Firebase.

Goals
- LVGL-based modular UI; FreeRTOS tasks for concurrency; Firebase for remote triggers.

Quickstart
- Open in PlatformIO; configure `platformio.ini` board and `lib_deps` for LVGL + Firebase.

Structure
- See docs/STRUCTURE.md for the recommended file layout and responsibilities.
